# JP Website (draft)

A personal portfolio website created to showcase my projects, technical skills, and journey as a developer. This website serves as a central hub for visitors to learn more about me and explore the work I've built.

## Features

- Responsive multi-section layout
- Project showcase and highlights
- About Me section
- Skills and technologies overview
- Contact information
- Clean and user-friendly navigation

## Technologies Used

- HTML
- CSS
- JavaScript

## Getting Started

### Clone the repository

```bash
git clone https://github.com/jayperkins150/jp-website.git
```

### Run locally

Open `index.html` in your browser.
